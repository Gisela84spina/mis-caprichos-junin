# Tienda Digital

Plataforma de catálogo online para pequeños comercios que necesitan
mostrar productos de forma clara y recibir pedidos ordenados vía WhatsApp.

## Objetivo
Diseñar un sistema reutilizable y adaptable a distintos rubros
(pastas, rotiserías, tiendas barriales),
priorizando simplicidad, velocidad y experiencia mobile.

## Alcance del proyecto
- Home con productos destacados y categorías
- Estructura pensada para promos y más vendidos
- Flujo de armado de pedido sin fricción
- Integración con WhatsApp Business
